# sunspot.academy

## Installation

* Prepare Python Environment
    ```
    pipenv install --dev
    pipenv shell
    ```
* Prepare JavaScript Environment
   ```
   yarn install
   ```
* Start Backend Server
   ```
   python run.py
   ```
* Start Webpack Server
   ```
   yarn serve
   ```


## Deploy

This website is currently hosted on Heroku.

* Create new app on heroku and connect the repo
